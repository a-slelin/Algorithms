# Задачи с Acmp

## Замечания

* Решения задач нумеруются в том же порядке, в котором они выставлены на сайте https://acmp.ru.
* Для нумерации в проекте используется нумерация с сайта. Пакет называется по паттерну **taskNNNN**, где NNNN - номер
  задачи (заполняется ведущими нулями при необходимости). Например, если решается задача 11, то пакет называется
  task0011.
* Описания задач находятся в папке с ресурсами под соответствующим номером задачи.
* Тесты для задач берутся также с сайта.
