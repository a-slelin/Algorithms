# Алгоритмы

Этот репозиторий содержит реализацию различных алгоритмов на Java.
Цель проекта — демонстрация работы алгоритмов, практическое применение, сравнение эффективности.
Также репозиторий предполагает решение алгоритмических задач с популярных площадок вроде LeetCode.

## Содержание:

Проект разбит на модули:

* **Sort** - реализации алгоритмов сортировки, включая пузырьковую, быструю, слиянием и др. Поиск порядковых статик.
* **LeetCode** - решение задач с площадки LeetCode.
* **Acmp** - решение задач с площадки Acmp (https://acmp.ru).
* **Tbank** - задачи, предложенные Т-банком в качестве стажировочных.
* **DynamicProgramming** - задачи по курсу "Динамическое программирование".
